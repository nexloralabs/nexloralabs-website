- **Custom Software Development**: Web apps, mobile apps, and SaaS platforms.

## 🛠 Tech Stack

- **Frontend**: Next.js 15+, Tailwind CSS, Framer Motion, Lucide React.
- **Performance**: Optimized images (WebP), Turbopack, and semantic HTML.
- **Forms**: Web3Forms integration.
- **Booking**: Cal.com integration.

## ⚡ Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🌐 Deployment

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com/new).
